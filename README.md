
# FOT Calculator MVP

Минимальная рабочая версия калькулятора ФОТ.

## Что это
Веб-приложение для расчета зарплат сотрудников на основе дневной выручки,
процентных ставок, часов работы, премий, боя и инвентаризации.

## Текущий статус
- Рабочая версия для Tilda перенесена в `index.html`
- Следующий этап — миграция в Next.js + Supabase
- Планируется публичный read-only доступ по ссылке
- Редактирование только после авторизации

## Как запустить локально
Просто открой `index.html` в браузере.

2. Подключить Supabase (Auth + Postgres)
